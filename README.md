# Smart Calculator in Python

A beginner-friendly Smart Calculator built using Python.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Square
- Cube
- Power
- Percentage

## Error Handling

This calculator can handle:

- Invalid number input (ValueError)
- Division by zero (ZeroDivisionError)
- Invalid operation selection

The program does not crash and allows the user to try again.

## Concepts Used

### Python Fundamentals
- Variables
- User Input

- Loops (while)

### Error Handling
- try

- ValueError

## Project Structure

Functions created:

- add()
- sub()
- multiply()
- divide()
- modulas()
- Square()
- percentage()
- power()
- cube()

Main calculator logic runs inside a `while True` loop until the user chooses to exit.

## Example Usage

Enter the first number: 10

Enter the second number: 5

Enter the operation (+, -, *, /, %, Square, Percentage, Power, Cube): *

Result: 50

Do you want to continue? (yes/no): no

Calculator closed
